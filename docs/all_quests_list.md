## 1. クエスト進行の特殊仕様

### 1.1 経過日数の成功・失敗差異
本作のクエストは、目標を達成して「成功（SUCCESS）」した場合と、全滅・ギブアップ・護衛対象死亡などで「失敗（FAILURE）」した場合とで、経過する日数（`time_cost`）が異なる設計になっています。
- **成功（SUCCESS）**: 本来の旅程日数（例: 8〜10日）が経過します。
- **失敗（FAILURE）**: 途中で敗退・帰還するため、経過する日数が少なくなります（例: 3〜5日）。

### 1.2 クエストによる強制拠点移動
メインシナリオの一部のクエストは、達成時または進行中にパーティの現在位置（拠点）が強制的に別の場所へと遷移します。
- **第5話「大義という名の虚妄」クリア時**: 聖帝国から夜刀神国の「最果ての村（`loc_frontier_village`）」へ強制的に移動します。
- **第10話「世界の底が抜ける日」クリア時**: 夜刀神国から華龍神朝の「開拓村（`loc_karyu_village`）」へ強制的に移動します。
- **第15話「天軍の長」クリア時**: 華龍神朝から聖帝国の「王都レガリア（`loc_regalia`）」へ強制的に移動します。
