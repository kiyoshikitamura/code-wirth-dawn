## 第1章：世界観と4大国家

### この世界について

かつて一つだった大陸は、四つの根源的な思想——**秩序・混沌・正義・悪**——をめぐる果てしない争いによって四つの国家に分裂した。

それぞれの国は己の理想こそが世界の真理だと信じ、覇権を競い合っている。国境の町は支配者が変わるたびに荒廃と繁栄を繰り返し、民は翻弄され続ける。

あなたはどの国にも属さない**旅人**。剣と知恵を頼りに、この混沌とした世界を渡り歩く冒険者だ。

---

### 4大国家

#### ローランド聖王国（秩序）
白亜の城壁と尖塔が並ぶ、中世ヨーロッパ風の騎士の国。法と信仰による統治を理想とし、聖騎士団が秩序を守る。首都**王都レガリア**では大聖堂の祝福を受けることができる。

> 特色：回復・防御系スキルが豊富。聖水、祝福の護符などの信仰系アイテム。

#### 砂塵の王国マルカンド（混沌）
灼熱の砂漠に栄える交易と錬金術の国。自由競争と実力主義が支配し、商人も盗賊も等しく力で道を切り開く。首都**黄金都市イスハーク**は世界最大の市場を擁する。

> 特色：毒・罠・回避系スキル。香辛料、錬金術キットなど独特のアイテム群。

#### 夜刀神国（正義）
神話と義を重んじる、中世日本風の島国。呪術と剣術が共存し、忍びの者が暗躍する。首都**神都「出雲」**は桜と社に包まれた神聖な都。

> 特色：居合・忍術・呪術系スキル。呪符、忍玉など和風アイテム。

#### 華龍神朝（悪）
力と実利による覇道を掲げる、中世中華風の大国。拳法家と策士が跋扈し、弱肉強食の法則が全てを支配する。首都**天極城「龍京」**は険しい岩峰の上にそびえる。

> 特色：連撃・気功・暗殺系スキル。龍井茶、護法符など東洋系アイテム。

## 第2章：はじめての冒険

### キャラクター作成

ゲームを開始すると、旅人の名前を入力するだけで冒険が始まる。
難しい設定は不要——あなたの物語は、最初の一歩から綴られていく。

### 基本ステータス

| ステータス | 説明 |
|---|---|
| **HP** | 体力。0になると戦闘不能。宿屋や回復アイテムで回復する |
| **ATK** | 攻撃力。武器やアクセサリの装備で上昇する |
| **DEF** | 防御力。防具やアクセサリの装備で上昇する |
| **Level** | 経験による成長。戦闘勝利で経験値を得てレベルが上がり、基礎ステータスが強化される |
| **Vitality** | 寿命。この世界で最も貴重なもの。行動を重ねると少しずつ減り、0になると旅は終わる |
| **Gold** | 所持金。装備やアイテムの購入、移動費用、酒場での傭兵雇用などに必要 |

### 宿屋（ホーム）

宿屋はあなたの活動拠点。ここから世界へ旅立ち、ここへ帰ってくる。

- **休息**：HPを全回復する
- **ギルド**：冒険者ギルドの依頼ボード。難易度別（Easy / Normal / Hard）に依頼が並ぶ
- **道具屋**：アイテムやスキルの購入・売却
- **酒場**：噂話を聞いたり、傭兵を雇う
- **所持品**：インベントリの確認と装備変更
- **旅立つ**：ワールドマップを開き、次の目的地へ

### 最初にやるべきこと

1. 道具屋で**傷薬**と基本的な**武器・防具**を購入する
2. 酒場で世界の噂を聞き、情報を集める
3. 近くの拠点へ移動し、道中の敵と戦って経験を積む

## 第3章：戦闘の仕組み

### カードバトルの基本

戦闘はターン制の**カードバトル**で行われる。

毎ターン、手札から使いたいカードを選び、APを消費して発動する。敵も同様にカードで行動する。敵のHPを0にすれば勝利、あなたのHPが0になれば敗北だ。

### AP（行動ポイント）

APは1ターンに使える行動力の上限。強力なカードほど多くのAPを必要とする。

- 毎ターン開始時にAPが回復する
- APが足りないカードは使えない
- 弱いカードを複数使うか、強いカードを1枚使うか——戦略の要

### カードの種類

| 種類 | 説明 |
|---|---|
| **攻撃 (Skill)** | 敵にダメージを与えるカード。単体攻撃や全体攻撃がある |
| **防御 (Defense)** | 受けるダメージを軽減する。守りを固めたいターンに |
| **回復 (Heal)** | HPを回復する。長期戦の生命線 |
| **魔法 (Magic)** | 魔法ダメージ。防御を無視する効果を持つものも |
| **支援 (Support)** | ATK上昇、状態異常解除など、直接ダメージ以外の効果 |

### 状態異常

戦闘中、カードやアイテムの効果で様々な状態異常が発生する。

| 状態 | 効果 |
|---|---|
| **毒** | ターン終了時にHPが減少し続ける |
| **炎上** | 毒と同様の持続ダメージ。火炎瓶などで付与 |
| **出血** | カードを使用するたびに追加ダメージ |
| **スタン** | 一定ターン行動不能になる |
| **ATK UP / DOWN** | 攻撃力が一時的に増減する |
| **DEF UP / DOWN** | 防御力が一時的に増減する |

### 戦闘アイテム

手札のカードとは別に、所持しているアイテムを戦闘中に使用できる。傷薬でHPを回復したり、爆弾で敵全体にダメージを与えたり、煙玉で逃走を試みることもできる。

### 勝利と敗北

- **勝利**：経験値とゴールド、まれにアイテムを獲得する
- **敗北**：ペナルティを受けて拠点に帰還する

## 第4章：装備と強化

### 3つの装備カテゴリ

装備品はインベントリから装着するだけでステータスが恒久的に上昇する。

| カテゴリ | 効果 | 例 |
|---|---|---|
| **武器** | 主にATKが上昇 | ショートソード、青龍偃月刀、神器:草薙 |
| **防具** | 主にDEFが上昇 | 革鎧、聖騎士の全身鎧、龍鱗の鎧 |
| **アクセサリ** | ATK/DEF/HPなど多彩な補正 | 冒険者の靴、幸運のコイン、錬金術セット |

### 各国の装備

各国家には独自の装備が存在する。旅先の道具屋でしか手に入らないものも多い。

- **ローランド聖王国**：騎士の盾、白銀の槍、聖騎士の全身鎧
- **マルカンド**：シミター、魔人の曲刃、王宮の絹衣
- **夜刀**：大弓、当世具足、鬼武者の鎧
- **華龍**：青龍偃月刀、鉄の爪、龍鱗の鎧

### 呪い装備

一部の装備には強力な効果と引き換えに、重いデメリットが伴う。

- **呪いの仮面**：ATK+4だがDEF-4。攻撃に全てを捧げる覚悟が要る
- **般若の面**：ATK+6だが最大HP-20。力を得る代わりに体が蝕まれる
- **呪いの偶像**：ATK+1だがDEF-2。触れてはいけなかったもの

### ボスドロップ限定装備

道具屋では手に入らない、強敵を倒した者だけが手にできる装備が存在する。通常の市販品を遥かに凌ぐ性能を誇る。

## 第5章：スキルカード

### スキルカードとは

道具屋で販売されている**教本**や**巻物**を購入すると、新たなスキルカードを習得できる。習得したスキルカードはデッキに組み込み、戦闘で使用する。

### デッキコスト

デッキに入れられるカードの総量には**コスト上限**がある。強力なカードほどコストが高い。限られた枠に何を入れるかが、戦略の鍵となる。

### 各国固有スキル

| 国 | 代表スキル |
|---|---|
| ローランド聖王国 | 聖剣（大ダメージ）、祈り（全体回復）、聖壁（全体バリア） |
| マルカンド | 毒刃（毒付与）、蜃気楼（全体回避UP）、砂の罠（拘束） |
| 夜刀 | ツバメ返し（高威力）、影縫い（スタン）、居合切り（最強の一撃） |
| 華龍 | 連撃（連続攻撃）、鉄布衫（重防御）、龍の咆哮（全体ATKダウン） |

### 闇市の禁術

崩壊した拠点の闇市では、通常では手に入らない**禁断のスキル**が売られている。絶大な力を持つが、代償も大きい。

## 第6章：アイテムと道具屋

### 消耗品

戦闘中やフィールドで使い切りの効果を発揮するアイテム。

| アイテム | 効果 |
|---|---|
| 傷薬 / 高級傷薬 | HPを回復 |
| 解毒剤 | 毒状態を解除 |
| 火炎瓶 | 敵にダメージ+炎上 |
| 煙玉 | 70%の確率で逃走 |
| 忍玉 | 確実に逃走 |
| 簡易テント | フィールドでHPを50%回復 |
| 大型爆弾 | 敵全体にダメージ |

各国にはその地域限定の消耗品もある。龍井茶（HP回復+毒解除）、大聖堂の祝福（HP回復+バフ）など、旅先でしか手に入らない品々を見逃すな。

### 交易品

使用効果はないが、売却することでゴールドに換える品物。敵から奪ったり、報酬として得ることがある。

### 道具屋の品揃え

道具屋の陳列は**訪れるたびに変化する**。全ての商品が常に並ぶわけではなく、カテゴリごとにランダムで選ばれた品が棚に並ぶ。欲しいものがなければ、時間をおいてまた訪れよう。

### 物価変動

アイテムの価格は拠点の**繁栄度**によって変動する。

| 繁栄度 | 物価 |
|---|---|
| 絶頂・繁栄 | 標準価格 |
| 停滞 | やや高い（×1.2） |
| 衰退 | 高い（×1.5） |
| 崩壊 | 闇市のみ（×3.0） |

安く仕入れて高く売る——商人の知恵も冒険者の武器だ。

## 第7章：旅と移動

### ワールドマップ

世界には**20の拠点**が点在し、街道で結ばれている。ワールドマップから行き先を選び、旅に出る。

### 移動のルール

- **隣接する拠点にのみ移動できる**。遠くの拠点に行くには、いくつもの拠点を経由する必要がある
- 移動には**日数**と**移動費（ゴールド）**がかかる
- 日数が経過するとキャラクターは歳を重ね、Vitalityが消費される

一歩一歩の旅こそが、この世界の醍醐味だ。

### 道中の危険

移動中、**ランダムに敵と遭遇する**ことがある。万全の準備なく旅立てば、道半ばで倒れかねない。

### 首都への入場

4大国家の首都は特別な場所であり、入場には**通行許可証**が必要。許可証は各地の道具屋で購入できる（20,000G）。

## 第8章：酒場と仲間

### 酒場でできること

拠点の酒場は情報と出会いの場。

#### 噂話を聞く

酒場の客たちから**世界の噂**を聞くことができる。世界情勢の変化、近くの危険な敵、珍しいアイテムの情報など——旅を有利に進めるヒントが得られる。

#### 傭兵を雇う

酒場にいるNPC（傭兵）を**雇って仲間にする**ことができる。仲間は戦闘で共に戦い、あなたを助けてくれる。

- 傭兵にはそれぞれ固有のステータスとスキルがある
- 雇用には契約金（ゴールド）が必要

#### 英霊の招集

**他のプレイヤーの旅が終わった後**、そのキャラクターは「英霊」として酒場に現れることがある。英霊を招集するには高額の契約金が必要だが、歴戦の強者の力を借りることができる。

## 第9章：世界の変遷と名声

### 繁栄度 — 世界は生きている

各拠点には**繁栄度**（5段階）があり、世界中のプレイヤーの行動によって変動する。

| 段階 | 状態 | 影響 |
|---|---|---|
| **絶頂** | 最も栄えている | 戦闘開始時に市民からの支援を受けられる |
| **繁栄** | 安定した状態 | 通常の冒険環境 |
| **停滞** | わずかな不穏 | 物価がやや上昇 |
| **衰退** | 治安の悪化 | 物価上昇、戦闘に不利な条件が加わる |
| **崩壊** | 統治の崩壊 | 通常の店は閉鎖、闇市のみ営業。戦闘が著しく不利に |

繁栄度は**6時間ごと**に更新される。あなたの行動も、この世界の命運に影響を与えている。

### 名声

各拠点であなたがどう見られているかは、**名声**として記録される。

- **英雄**：拠点の住民から敬われ、様々な恩恵を受ける
- **よそ者**：特に良くも悪くもない初期状態
- **ならず者**：警戒され、取引を拒否されることがある
- **犯罪者**：**出禁**。その拠点では一切の取引ができなくなる

名声が極端に低い状態で街道を歩けば、**賞金稼ぎに襲われる**危険もある。

名声のリセット手段は存在するが……その代償は計り知れない。

## 第10章：死と継承

### Vitality — 限りある命

この世界で最も貴重な資源、それが**Vitality（寿命）**だ。

- 拠点間の移動や特定の行動でVitalityは少しずつ減少する
- Vitalityが0になると、あなたの旅は**終わりを迎える**
- 通常の手段ではVitalityは回復しない

ただ一つ、伝説の**竜血**——竜の血液から精製される極めて希少な霊薬だけが、失われた寿命を取り戻すことができるという。

### 引退と継承

Vitalityが尽きた時、あるいは自らの意志で引退を選んだ時、あなたの冒険者としての人生は幕を閉じる。

しかし、全てが失われるわけではない。

次の世代——新たな旅人に、**遺産を託すことができる**。

- **ゴールド**：上限つきで次代に引き継がれる
- **形見アイテム**：わずかな数の装備やアイテムを遺品として残せる
- **クエスト記憶**：メインストーリーのクリア記録は永遠に刻まれる。ただし、それ以外のクエストは新たな旅人には未知の依頼として再び現れる

先代の遺した剣を手に、新たな旅人が立ち上がる。死は終わりではない。それは次の物語の始まりだ。

### 闇市の禁術

崩壊した拠点の闇市には、道徳を踏み越えた者だけがたどり着ける品々がある。

- **禁術の秘薬**：HP全回復に加え、毒解除やバフまで得られる万能薬。ただし高額
- **帳簿の改竄**：全拠点での名声を白紙に戻す禁忌の手段。代償は莫大なゴールド
- **竜血**：唯一のVitality回復手段。その価格は常人が一生かけても稼げない額

光の道を歩むか、闇に手を染めるか——選ぶのはあなた自身だ。
