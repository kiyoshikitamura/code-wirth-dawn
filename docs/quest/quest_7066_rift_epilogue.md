## 1. クエスト概要

### 短文説明
```
[報告] 次元の深淵を閉じ、世界の危機を救ったことをギルドへ報告せよ。
```

### 長文説明
```
ついに破壊神サタンは滅び、次元の狭間から漏れ出ていた瘴気は完全に霧散した。
急激に収縮していく迷宮の崩壊から九死に一生を得て生還した私たちは、その足で冒険者ギルドへと向かう。
そこには、世界の崩壊を免れたことに安堵し、心からの感謝を捧げるバルガスとギルドの人々が待っていた。
世界の救世主となったプレイヤーたちの、栄光のエピローグ。
```

## 4. ノード詳細（10ノード）

#### `start_prep`（text）
**演出:** bg: bg_office
```text
大仕事を終え、重い足取りでギルドの扉を開ける。しかし、中に入った瞬間、割れんばかりの歓声と拍手が私たちを包み込んだ。
```

#### `start`（text）
**演出:** bg: bg_office
```text
ギルド内の冒険者たちが道を空ける。その先には、いつもの厳格な仮面をかなぐり捨て、満面の笑みを浮かべたギルド長バルガスが立っていた。
```

#### `text_vargas_01`（text）
**演出:** bg: bg_office, speaker: ギルド長バルガス
```text
「おお、帰ったか！ 次元の狭間が完全に閉じられたのを、魔導観測班が確認したぞ。サタンを……あの破壊神を本当に討ち倒したのだな！」
```

#### `text_vargas_02`（text）
**演出:** bg: bg_office, speaker: ギルド長バルガス
```text
「お前たちが成し遂げたのは、ただの魔物討伐ではない。崩壊の危機にあった世界そのものを救い出したのだ。ギルドを代表して、いや、この世界のすべての人々を代表して、心から感謝する！」
```

#### `text_vargas_03`（text）
**演出:** bg: bg_office
```text
バルガスは私たちの肩を力強く叩き、大きな勲章と報酬を手渡した。周囲の冒険者たちも、口々に私たちの名を叫び、称賛を惜しまない。
```

#### `world_peace`（text）
**演出:** bg: bg_office
```text
世界の中心にあった巨大な奈落はふさがり、空には再び穏やかな陽光が戻っていた。世界の均衡は保たれ、また新しい一日が始まる。
```

#### `epilogue_end`（text）
**演出:** bg: bg_office
```text
しかし、私たちの冒険が終わったわけではない。この世界にまだ見ぬ危機や謎がある限り、私たちは再び剣を取り、歩み続けるだろう。
（※このノードをもってクエストクリアとなり、エピローグが幕を閉じる）
```
