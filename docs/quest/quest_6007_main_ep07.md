## 1. クエスト概要

### 短文説明
```
夜刀の関所。死んだ兵士が託した機密書を手に、入国の道を切り拓け。
```

### 長文説明
```
夜刀神国の国境関所に辿り着いた。
だが通行証のない流れ者に、門は開かない。
手元には道中で死んだローランド兵から託された機密書がある。
——売って安全を買うか、実力で道を拓くか。
```
