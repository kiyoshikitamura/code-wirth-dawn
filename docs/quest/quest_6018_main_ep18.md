## 1. クエスト概要

### 短文説明
```
龍京沖の無人島に眠る石碑。英霊の記憶の中で、軍神アレスが待ち受ける。
```

### 長文説明
```
英霊の導きに従い、龍京の港から船で沖合の無人島へ渡った。
潮風と密林に囲まれた島の奥に、三つ目の石碑がある。
石碑に触れると、英霊が戦場で死闘を繰り広げた記憶が流れ込む。
記憶の中で待ち受けるのは——軍神アレス。
```
