## 1. クエスト概要

### 短文説明
```
交易路で商人が消えている。王国が放った追跡者の正体を暴け。
```

### 長文説明
```
オアシス都市での毒物事件から数日。
マルカンドの交易路で有力商人が次々と行方不明になる事件が発生。
犯人は「賞金稼ぎ」を名乗る者たちだが、
その背後にはローランド王国の影がちらついていた。
```
