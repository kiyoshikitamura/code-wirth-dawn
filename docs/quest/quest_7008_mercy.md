## 1. クエスト概要

### 短文説明
```
[救援] 癒やし草を採取して傷ついた難民の野営地へ届ける。
```

### 長文説明
```
「難民の野営地で薬が尽きかけている。森で癒やし草を集めて届けてやってくれ。金はないが、恩は返す」
```

## 3. シナリオノード構成

### 全体フロー

```text
start
  └─[続ける]→ meet_sister
       └─[続ける]→ visit_camp
            └─[続ける]→ depart_forest
                 └─[続ける]→ gather_01
                      └─[続ける]→ gather_02
                           └─[続ける]→ gather_03
                                └─[続ける]→ gather_complete
                                     └─[届ける]→ deliver_herbs
                                          └─[続ける]→ end_success
```

> **注:** `gather_loop(5回)` は演出上3回の採取ノード + テキスト補完で5回分を表現。
> 実装上は `gather_loop` タイプで回数を指定する。

### ノード詳細

#### `start`（type: text）
**BGM**: `bgm_quest_calm`
**背景画像**: `bg_guild`
**SE**: `se_quest_accept`

**テキスト:**
```
掲示板に貼られた依頼書は、他のものとは紙の質が違っていた。
安い紙に、震える字で書かれている。

「薬草を集めて、野営地の修道女に届けてください。
　報酬はほとんど出せません。すみません」

金にならない仕事だ。だが——掲示板に残っているということは、
誰も手をつけていないということだ。
```

---

#### `meet_sister`（type: text）
**背景画像**: `bg_tavern_day`

**テキスト:**
```
依頼主に会いに行くと、酒場の片隅で修道女が待っていた。
白い法衣は薄汚れ、目の下に深い隈がある。

「来てくださったのですか……ありがとうございます」

彼女は小さく頭を下げた。
「私はシスター・エレナ。街の東にある難民の野営地で、
　怪我人の手当てをしています。ですが、薬草が底をついてしまって」

声は落ち着いているが、指先が細かく震えていた。
```

---

#### `visit_camp`（type: text）
**BGM**: `bgm_quest_sad`
**背景画像**: `bg_road_day`

**テキスト:**
```
エレナに案内されて、野営地を訪れた。

粗末なテントが並んでいる。毛布は足りていない。
子供たちが泥だらけの顔で、ぼんやりとこちらを見ている。

テントの奥では、熱にうなされる老人が横たわっていた。
布を巻いた腕から、染みた血が滲んでいる。

「あの方は先週の戦で逃げてきた人です。矢傷が化膿しています。
　癒やし草があれば……せめて、痛みだけでも」

エレナの目に涙はなかった。泣き尽くしたのだろう。
```

---

#### `depart_forest`（type: text）
**BGM**: `bgm_quest_calm`
**背景画像**: `bg_forest`

**テキスト:**
```
野営地を後にし、森の外れに向かった。
癒やし草の群生地があると、エレナが教えてくれた場所だ。

「沢の近く、苔の生えた岩の周りに生えています。
　白い小さな花が目印です。五束は必要です」

森に入った。木漏れ日の下、湿った土の匂い。
しばらく探索が必要になりそうだ。
```

---

#### `gather_01`（type: text / gather）
**背景画像**: `bg_forest`

**テキスト:**
```
沢沿いを歩くと、苔むした岩の陰に白い花を見つけた。
葉を摘むと、清涼な香りが指先に残る。これが癒やし草だ。

丁寧に根元から採取する。一束、二束。
まだ足りない。もう少し奥を探そう。
```

---

#### `gather_02`（type: text / gather）
**背景画像**: `bg_forest`

**テキスト:**
```
倒木の影にも群生を見つけた。三束目。

採取しながら、野営地の老人の顔を思い出す。
あの傷では、遅れれば命に関わる。急がねば。
```

---

#### `gather_03`（type: text / gather）
**背景画像**: `bg_forest`

**テキスト:**
```
沢の上流まで足を伸ばした。
岸辺の一角に、まとまった群生を発見。四束目、五束目。

規定の数が揃った。
布で丁寧に包む。傷んだら効果が落ちるからだ。
野営地へ急いで戻ろう。
```

---

#### `gather_complete`（type: text）
**背景画像**: `bg_forest`

**テキスト:**
```
五束の癒やし草を集め終えた。
日が傾き始めている。野営地まで、急ぎ足で一刻ほど。

腰に括りつけた布包みが揺れる。
たかが草一束だ。されど——これで誰かの命が繋がる。
```

---

#### `deliver_herbs`（type: check_delivery）
**BGM**: `bgm_quest_calm`
**背景画像**: `bg_road_day`

**テキスト:**
```
野営地に戻ると、エレナが入口で待っていた。
癒やし草の包みを渡すと、彼女は両手で受け取り、深く息をついた。

「これで……これで助けられます。ありがとう、本当に」

すぐに煎じ薬の準備に取りかかった。
手際がいい。慣れているのだ——こういう状況に。

「報酬は……ほとんど出せないのですが」
エレナが申し訳なさそうに、小さな革袋を差し出した。
中には銅貨が数枚。

「金はいい」と言おうとして、やめた。
彼女の誇りを傷つけたくなかった。
```
**params:**
```
type:check_delivery, item_slug:item_healing_herb, quantity:5, next:end_success
```

---

#### `end_success`（type: end, result: success）
**SE**: `se_quest_success`

**テキスト:**
```
薬草の納品完了。報酬は銅貨10枚——腹は膨れない。
だが、テントの奥から老人の呻き声が小さくなったのが聞こえた。

エレナが振り返り、遠くから深々と頭を下げた。

帰り道。いつもの酒場が見えた頃、
不思議と足取りが軽くなっていることに気づいた。
```

---

#### `end_failure`（type: end, result: failure）

**テキスト:**
```
薬草を集めきれなかった。
野営地のエレナに伝えると、彼女は微笑んだ。

「大丈夫です。きっと別の方が来てくれます」

その笑顔が、何より辛かった。
```
