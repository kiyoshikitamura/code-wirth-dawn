## 1. クエスト概要

### 短文説明
```
出雲の山奥に眠る石碑。英霊の記憶の中で、女神アルテミスが待ち受ける。
```

### 長文説明
```
英霊の導きに従い、出雲の山奥にある秘境へ向かった。
霧に包まれた山道を登り、人跡未踏の谷の奥に四つ目の石碑がある。
石碑に触れると、英霊が山道で繰り広げた死闘の記憶が流れ込む。
記憶の中で待ち受けるのは——女神アルテミス。
```
