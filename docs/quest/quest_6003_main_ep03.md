## 1. クエスト概要

### 短文説明
```
オアシスの井戸に毒。工作員を追跡し、戦争の火種を断て。
```

### 長文説明
```
砂塵王国マルカンドのオアシス都市。平和に見えるこの地に潜む間諜の影。
井戸に毒が投じられ、街がパニックに陥る。
犯人を追い詰めた先で明かされるのは、
「聖戦」の名を借りた開戦工作の真実だった。
```
