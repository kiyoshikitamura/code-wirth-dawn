## 1. クエスト概要

### 短文説明
```
[探索] 廃墟の金庫から物資を回収する。届けるか、売りさばくか。
```

### 長文説明
```
「先日の暴動で焼け落ちた区画に、まだ金庫が残っているはずだ。火事場泥棒どもより先に中身を回収してくれ」
```

---

## 2. 報酬定義

**正規ルート（届ける）:**
```
Gold:250|Rep:2
```

**闇ルート（売る）:**
```
Gold:500|Rep:-5|Evil:10|Chaos:10
```

## 3. シナリオノード構成

### 全体フロー

```text
start
  └─[続ける]→ enter_ruins
       └─[続ける]→ battle_01_intro
            └─[戦う]→ battle_01
                 ├─ [勝利] → deeper
                 │    └─[続ける]→ battle_02_intro
                 │         └─[戦う]→ battle_02
                 │              ├─ [勝利] → find_vault
                 │              │    └─[金庫を開ける]→ get_item
                 │              │         └─[続ける]→ choice_delivery
                 │              │              ├─[ギルドに届ける（正規）]→ deliver_guild → end_success_good
                 │              │              └─[闇市で売りさばく（闇）]→ sell_black → end_success_dark
                 │              └─ [敗北] → end_failure
                 └─ [敗北] → end_failure
```

### ノード詳細

#### `start`（type: text）
**BGM**: `bgm_quest_calm`
**背景画像**: `bg_guild`
**SE**: `se_quest_accept`

**テキスト:**
```
冒険者ギルドの掲示板から依頼書を剥がした。
焦げた紙の端に、走り書きで住所と金庫の位置が記されている。

「ああ、それか」とカウンターの担当が顔を上げた。
「先日の暴動で焼けた商店街の地下だ。持ち主は行方不明。
　金庫の中身を回収して届ければ、見つけ出すための費用に充てるとさ」

担当は肩をすくめた。
「火事場泥棒が先にたどり着く前に頼むよ。もう何人か目をつけてるらしい」
```
