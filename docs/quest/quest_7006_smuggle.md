## 1. クエスト概要

### 短文説明
```
[密輸] 中身不明の禁制品を隣街まで運ぶ。追手に注意。
```

### 長文説明
```
「この木箱を運ぶだけだ。中身は聞くな。隣街まで往復、金は弾む。ただし見つかったら自力で何とかしろ」
```

## 2.5. 失敗ペナルティ（共通仕様）

| 条件 | ペナルティ |
|------|-----------|
| クエスト失敗（敗北/撤退/ギブアップ） | 名声 -3〜-10（ランダム、現在拠点） |
| バトル敗北（全般） | VIT -1（クエスト/ランダムエンカウント/賞金稼ぎ共通） |
| 経過日数 | クエスト定義の days_failure 値を適用 |

> 備考: クエスト失敗時は結果画面でペナルティが表示され、確認後に拠点へ帰還する。バトル敗北のVIT -1はクエスト固有ではなくシステム共通のペナルティであり、マップ移動中のランダムエンカウントや賞金稼ぎ戦でも同様に適用される。特別な指定があるクエストはこの限りではない。

## 3. シナリオノード構成

### 全体フロー

```text
start
  └─[続ける]→ receive_cargo
       └─[続ける]→ depart
            └─[続ける]→ travel_smuggle (random_branch)
                 ├─ [50%: 追手出現] → checkpoint
                 │    └─[逃げる]→ chase → battle_hunter
                 │         ├─ [勝利] → escape_success → delivery_point
                 │         └─ [敗北] → end_failure
                 └─ [50%: 無事通過] → delivery_point
                      └─[続ける]→ handoff → end_success
```

### ノード詳細

#### `start`（type: text）
**BGM**: `bgm_quest_tense`
**背景画像**: `bg_tavern_night`
**SE**: `se_quest_accept`

**テキスト:**
```
裏路地の酒場。薄暗い奥の席に、フード越しに鋭い目が光っていた。

「来たか。座れ」

男——闇商人は、テーブルの下から油紙に包まれた木箱を押し出した。
「仕事は簡単だ。これを隣街まで運べ。往復八日。
　金は弾む。前金はない」

木箱の中から、かすかに薬品の匂いがする。
```

---

#### `receive_cargo`（type: text）
**背景画像**: `bg_tavern_night`

**テキスト:**
```
「中身は聞くな。それがルールだ」

闇商人はそう言って、煙管から煙を吐いた。
「もし関所で引っかかったら——そうだな、自力で何とかしろ。
　捕まっても俺の名前は出すなよ。出したところで、
　俺なんて人間はこの世に存在しないことになってるがな」

低い笑い声。冗談ではないらしい。

木箱を担いだ。見た目より重い。
何が入っているかは……考えないことにした。
```

---

#### `depart`（type: text）
**BGM**: `bgm_quest_tense`
**背景画像**: `bg_road_night`

**テキスト:**
```
人目を避け、夜明け前に街を出た。
街道を避けて脇道を選ぶ。闇商人の指示だ。

月明かりだけが頼りの道。木箱が背中にずしりと食い込む。
振り返ると、街の灯りが遠くに小さく光っていた。
もう引き返せない。
```

---

#### `travel_smuggle`（type: travel + random_branch）
**背景画像**: `bg_road_night`

**テキスト:**
```
道の半ば。空が白みかけた頃——背後から馬蹄の音が聞こえた。
```
**params:**
```
type:travel, dest:loc_border_town, days:4, gold_cost:0
random_branch: prob:50, next:checkpoint, fallback:delivery_point
```

---

#### `checkpoint`（type: text）
**BGM**: `bgm_battle_tense`
**背景画像**: `bg_road_day`

**テキスト:**
```
「そこで止まれ！」

騎馬の男が二人、前方に立ちはだかった。
揃いの革鎧。自警団か、あるいは——賞金稼ぎか。

「おい、その荷物を見せろ。最近、この辺りで禁制品の流通が増えてるんだ。
　中身を改めさせてもらうぞ」

荷を見られたら終わりだ。
```
**選択肢:**
| ラベル | 次ノード |
|--------|---------|
| 逃げる | `chase` |

---

#### `chase`（type: text）
**背景画像**: `bg_road_day`

**テキスト:**
```
木箱を抱えて脇道に飛び込んだ。
背後で怒声が上がる。

「逃げた！ 追え！ やっぱり怪しいぞ！」

茂みを掻き分け、崖沿いの獣道を走る。
追手の足音が近づいてくる。——もう逃げきれない。
ここで迎え撃つしかない。
```

---

#### `battle_hunter`（type: battle）
**BGM**: `bgm_battle_normal`

| 設定 | 値 |
|-----|-----|
| 敵グループ | `enemy_bandit_guard` x2 |
| 敵名 | 賞金稼ぎ / 自警団員 |
| 備考 | 中程度の強さ。攻撃的AI |

**params:**
```
type:battle, enemy_group_id:grp_bounty_pair, next:escape_success, fail:end_failure
```

---

#### `escape_success`（type: text）
**BGM**: `bgm_quest_tense`
**背景画像**: `bg_road_day`

**テキスト:**
```
追手を退けた。
倒れた男の懐から自警団の紋章が転がり落ちた。
正規の組織か……殺してはいないが、面倒なことになりそうだ。

荷物を確認する。無事だ。先を急ごう。
もうこの道は使えない。別のルートで目的地へ向かう。
```

---

#### `delivery_point`（type: text）
**BGM**: `bgm_quest_calm`
**背景画像**: `bg_tavern_night`

**テキスト:**
```
目的地に到着した。
指定された裏路地の奥で、無言の男が待っていた。

木箱を差し出すと、男は中身を手早く確認し、頷いた。
「ご苦労」

それだけだった。名前も、礼も、何もない。
```

---

#### `handoff`（type: text）
**背景画像**: `bg_tavern_night`

**テキスト:**
```
帰り道。闇商人から伝言が入った。
「上出来だ。報酬は約束通り。次の仕事もあるぞ」

重い金袋を懐に収めた。
何を運んだのか——知らない方がいい。きっと、そうだ。
```

---

#### `end_success`（type: end, result: success）
**SE**: `se_quest_success`

**テキスト:**
```
密輸完了。法の目を潜り抜け、報酬を手にした。
名声は落ちた。だが金は手に入った。
この世界では、それが全てだと言う者もいる。
```

---

#### `end_failure`（type: end, result: failure）

**テキスト:**
```
追手に叩きのめされ、荷を奪われた。
闇商人からの連絡は二度と来なかった。
当然だ。使えない駒は切り捨てる——それがあの世界のルールだ。
```
