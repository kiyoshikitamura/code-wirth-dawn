## 1. クエスト概要

### 短文説明
```
古代神殿で世界の真実を知る。全ての戦争は、神による間引きだった。
```

### 長文説明
```
大名行列の襲撃、暗部の暗躍——
全てを裏で操る黒幕の痕跡を追い、禁域の古代神殿へ辿り着いた。
祭壇に映し出される真実。ローランドの大義も、マルカンドの防衛も、
夜刀の一揆も——全ては天上の神々が仕組んだ「間引きの遊戯」だった。
```
