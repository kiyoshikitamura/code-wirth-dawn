## 1. クエスト概要

### 短文説明
```
世界の中心に突如現れた巨大な空洞「狭間の迷宮」。その先遣調査へと向かえ。
```

### 長文説明
```
世界の地殻に突如出現した巨大な次元の空洞「狭間の迷宮」。
そこからは禍々しい悪魔の瘴気が漏れ出ており、世界の命運を脅かす恐れがあるという。
すでにギルドが派遣した腕利きの先遣隊が誰一人として戻っておらず、ギルド内には異様な緊張感が張り詰めていた。
私達はギルド長からこの極めて危険な迷宮の調査を託される。
大渓谷の奈落の底、ねじ切れた空間の入り口で、私達は己の決意と向き合うことになる。
また、迷宮から持ち帰った「宝珠」の波動は、さらなる深奥への道を示すことになるだろう。
```
