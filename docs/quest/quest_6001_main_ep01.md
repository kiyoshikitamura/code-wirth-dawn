## 1. クエスト概要

### 短文説明
```
辺境の輸送隊護衛。老騎士ガウェインの下で、最初の血を流せ。
```

### 長文説明
```
王国辺境の名もなき村。荒涼たる大地を渡る風に身を晒しながら、
一介の傭兵として生き抜く日々が始まる。
王国軍の輸送部隊護衛という最初の仕事が、
君を戦乱の渦中へと引きずり込むことになるとは、まだ知る由もなかった。
```

- **BGM**: `bgm_quest_calm` / **背景**: `bg_road_day`

**テキスト:**
```
肌を焦がすような乾いた風が、荒涼たる砂丘を撫でる。

ここはローランド聖王国の辺境。
かつて栄華を極めた王国の威光も、この村までは届かない。
崩れかけた土壁。痩せた家畜。乾いた井戸の底に溜まった泥水。
人々は日々の糧を得るのにも苦労している。

そんな場所から——君の物語は始まる。
```

**テキスト:**
```

「俺たちを雇ったのは、お前らの国の連中だ」
```
