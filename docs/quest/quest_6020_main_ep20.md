## 1. クエスト概要

### 短文説明
```
レガリアの遺跡に眠る最後の石碑。先代の記憶の中で、全能神ゼウスが待ち受ける。
```

### 長文説明
```
三つの遺産を手に、最後の石碑があるレガリア郊外の遺跡へ向かった。
石碑に触れると、英霊がガウェインと共に戦った最後の記憶が流れ込む。
記憶の中で待ち受けるのは——全能神ゼウス。
全ての遺産が揃っていれば力の加護を得られるが、
揃っていなければ、神はその全力を以て叩き潰しにくる。
```
