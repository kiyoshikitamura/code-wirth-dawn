## 1. クエスト概要

### 短文説明
```
出雲に響く大天使ガブリエルの「啓示」。人々が互いを敵と見なし始めた。
```

### 長文説明
```
龍京でラファエルを退け、夜刀の首都・出雲へ向かった。
だが出雲を襲っていたのは炎でも眠りでもなく——混乱だった。
大天使ガブリエルの「啓示」が人々の心に直接囁き、
隣人を敵と見なさせ、同士討ちを引き起こしている。
```
