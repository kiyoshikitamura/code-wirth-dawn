## 3. シナリオノード構成

### 全体フロー

```text
start
  └─[続ける]→ arrive_village
       └─[続ける]→ enter_forest
            └─[続ける]→ find_tracks
                 └─[続ける]→ discover_remains
                      └─[続ける]→ find_bear
                           └─[戦う]→ battle_bear
                                ├─ [勝利] → collect_pelt → end_success
                                └─ [敗北] → end_failure
```

### ノード詳細

#### `start`（type: text）
**BGM**: `bgm_quest_calm`
**背景画像**: `bg_guild`
**SE**: `se_quest_accept`

**テキスト:**
```
開拓村の村長が酒場に駆け込んできた。日に焼けた顔が、今日は土気色だ。

「傭兵さん、頼むよ。山の熊がまた降りてきたんだ。
　家畜が三頭やられた。柵なんか紙みたいに破られちまった」

男は声を落とした。
「先週、猟師のヤコブを出したんだが……戻ってこない。
　あんたなら——頼めるか？」

その目には、縋るような光があった。
```

---

#### `arrive_village`（type: text）
**背景画像**: `bg_road_day`

**テキスト:**
```
開拓村に着いた。小さな集落だ。
壊された柵の前で、子供が泣いている。母親がその頭を撫でていた。

村の男たちは遠くの山を見上げている。
誰も山に入ろうとしない。その気持ちは分かる。

村長が山道の入口を指差した。
「あの先の沢の辺りで目撃されてる。気をつけてくれよ……」
```

---

#### `enter_forest`（type: text）
**BGM**: `bgm_quest_tense`
**背景画像**: `bg_forest`

**テキスト:**
```
山道に入った。木々は鬱蒼と茂り、日差しが届かない。
足元の落ち葉が湿って滑る。

獣の匂いがする。それに混じって、かすかに血の臭い。
前任の猟師が通ったであろう道を辿る。
```

---

#### `find_tracks`（type: text）
**背景画像**: `bg_forest`

**テキスト:**
```
泥に深く刻まれた足跡を見つけた。
人の手のひらよりも大きい——相当な体格の個体だ。

爪痕が木の幹に残っている。かなり新しい。
近くにいる。
```

---

#### `discover_remains`（type: text）
**背景画像**: `bg_forest`

**テキスト:**
```
沢の手前で、散らばった荷物を見つけた。
猟銃が折れ、弾薬入れが踏み潰されている。ヤコブのものだろう。

少し離れた場所に、革のブーツが片方だけ転がっていた。
中に——見なかったことにした。

猟師でも敵わなかった相手だ。気を引き締めろ。
```

---

#### `find_bear`（type: text）
**背景画像**: `bg_forest`

**テキスト:**
```
沢のほとりで、それは待っていた。

立ち上がると人の背丈をゆうに超える巨体。
毛並みは荒れ、両目は充血している。口の周りが赤黒く染まっている。

冬ごもりに失敗した飢えた熊——いや、もはや人の味を覚えた獣だ。
こちらに気づくと、低く唸り声を上げた。逃げるつもりはないらしい。
```
**選択肢:**
| ラベル | 次ノード |
|--------|---------|
| 戦う | `battle_bear` |

---

#### `battle_bear`（type: battle）
**BGM**: `bgm_battle_normal`

| 設定 | 値 |
|-----|-----|
| 敵グループ | `enemy_giant_bear` x1 |
| 敵名 | 凶熊 |
| 備考 | HP高め・ATK高め。1体のボス型戦闘 |

**params:**
```
type:battle, enemy_group_id:grp_bear_solo, next:collect_pelt, fail:end_failure
```

---

#### `collect_pelt`（type: reward + text）
**BGM**: `bgm_quest_calm`
**背景画像**: `bg_forest`

**テキスト:**
```
熊を仕留めた。
最後の一撃で倒れた巨体は、地面を揺らした。

手際よく毛皮を剥ぐ。傷はあるが、しっかりした質だ。
高く売れるだろう——あるいは、防具の素材にもなる。

ヤコブの荷物を集めて、村に持ち帰ることにした。
せめて遺品だけでも。
```
**params:**
```
type:reward, item_slug:item_bear_pelt, quantity:1, next:end_success
```

---

#### `end_success`（type: end, result: success）
**SE**: `se_quest_success`

**テキスト:**
```
村に戻ると、村長が出迎えた。
毛皮を見せると、安堵の息を漏らした。

「これで安心して眠れる。本当にありがとう」

ヤコブの遺品を渡すと、村長の顔が歪んだ。
「……そうか。やっぱり、駄目だったか」

報酬と毛皮を受け取り、村を後にした。
背後で子供の笑い声が聞こえた。少なくとも、明日は平穏だろう。
```

---

#### `end_failure`（type: end, result: failure）

**テキスト:**
```
熊の剛腕に叩きのめされ、辛うじて山を下りた。
村に戻ると、村長は何も言わなかった。
ただ、壊れた柵の向こうで、子供がまた泣いていた。
```
