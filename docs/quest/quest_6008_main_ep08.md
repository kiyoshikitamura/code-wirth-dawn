## 1. クエスト概要

### 短文説明
```
真夜中の宿に暗殺者。闇の中で刃を凌ぎ、夜刀の裏社会の闇に触れろ。
```

### 長文説明
```
夜刀神国に入境して数日。辺境の宿屋で束の間の休息を取っていた矢先、
真夜中に黒装束の暗殺者が襲撃してきた。
異国の闇は深く、底知れないほど冷たい。
```
