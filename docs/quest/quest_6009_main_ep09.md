## 1. クエスト概要

### 短文説明
```
大名行列の護衛。だが荷の中身は年貢米——飢えた民が襲ってくる。
```

### 長文説明
```
暗殺者の襲撃を凌ぎ、腕の立つ傭兵として名が広まり始めた。
有力な大名からの直接指名で護衛依頼が舞い込む。
だが、護るべき荷の正体を知った時——
剣を向けるべき相手が、わからなくなる。
```

#### `text_procession`（type: text）
- **背景**: `bg_mountain`

**テキスト:**
```
依頼内容は、領地内を横断する大名行列の護衛。

表向きは「近隣の古社へ向けた祭礼行事の移動」と説明されている。
だが——行列の装備の量には違和感があった。

祭礼にしては、荷車が多すぎる。
しかも荷車の車輪の跡が、異常に深い。
——中身が、とてつもなく重いのだ。
```

何かを知っているのか。——それとも、知りたくないのか。
