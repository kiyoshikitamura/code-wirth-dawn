## 1. クエスト概要

### 短文説明
```
最強の大天使ミカエルがレガリアに降臨。ヴォルグと共に、最後の都を守り抜け。
```

### 長文説明
```
三体の大天使を退け、最後の標的であるローランドの首都レガリアへ急行した。
だが天軍の長たる大天使ミカエルは、他の三体とは格が違う。
圧倒的な武力でレガリアを蹂躙するミカエルに、
ヴォルグと共に最後の戦いを挑む。
```
