## 1. シナリオ世界観と進行コンセプト

### 1.1 背景
世界の中心地殻に突如として開いた謎の巨大空洞 ── 「狭間の迷宮」。かつて世界を脅かした伝説の悪の魔導士「ヴァルグナ」が創造したとされるその迷宮からは、おびたただしい数のデーモンが湧き出していた。ギルドから調査を命じられたプレイヤーは、未帰還となった先遣隊の足跡を追い、奈落の深部へと足を踏み入れる。

### 1.2 探索進行と通路開通フロー
メタ的な章切り替えではなく、「探索の結果、強固な鍵を開け、あるいは地下階段を発見して前進する」という一貫した探索フローをクエスト間の依存関係で表現する。

1.  **「導入」 (1F入り口)**: 迷宮の入り口周辺を巡り、デーモンの出現と迷宮の存在を確認しギルドに報告する。
2.  **「上層」 (1F〜3F)**:
    - 3F最奥でボス「狂乱のデーモン・アスタロト」を倒すことで、その背後にある「地下4Fへ続く鉄格子の鍵」を入手する。
    - 宿屋に帰還（クリア）後、鍵を開錠したことで「深層（4F）」の調査依頼が解放される。
3.  **「深層」 (4F〜7F)**:
    - 前作で開いた鉄格子の先（4F）から探索を開始。
    - 7F最奥でボス「深淵の魔獣ベヒモス」を倒し、「地下8Fへ続く奈落の石階段」を開通する。
    - 宿屋に帰還（クリア）後、階段を降りて「最深部（8F）」の調査依頼が解放される。
4.  **「最深部」 (8F〜10F)**:
    - 開通した奈落の石階段（8F）から攻略を開始。
    - 10F最奥で待つ「魔導士ヴァルグナ」との最終決戦に挑む。

## 2. アライメント結界ゲート (50%閾値判定 & 不合格明示)

「深層」および「最深部」のダンジョン内に、プレイヤーの性質を試すアライメント結界扉を配置する。属性確認UIがないことを補うため、扉を調べた際のメッセージで開かない理由をフレーバーテキストで明示する。

### 2.1 秩序・正義の結界 (5F隠し部屋前)
*   **判定条件**: プレイヤーの `order_pts` 割合 >= 50% かつ `justice_pts` 割合 >= 50% (アライメント合計値に対する比率)。
*   **合格時（突破可能）**:
    - メッセージ: 「貴方の魂に宿る秩序と正義の光が反応し、白銀の結界が静かに霧散した。奥へと進むことができる……」
    - 奥の部屋で貴重品 **「狭間の魔石・青（秩序・正義 of 証）」** を獲得。
*   **不合格時（閉鎖）**:
    - メッセージ: **「強い邪気と混沌の気配を感じる。秩序を重んじ、正義の道を歩む者でなければ、この聖なる結界を破ることはできないだろう……」** と表示し、進行を遮断する。

### 2.2 混沌・悪の結界 (9F生贄の祭壇前)
*   **判定条件**: プレイヤーの `chaos_pts` 割合 >= 50% かつ `evil_pts` 割合 >= 50%。
*   **合格時（突破可能）**:
    - メッセージ: 「貴方が抱く混沌の意思と悪徳の影に呼応し、血塗られた封印が嗤うように解き放たれた……」
    - 奥の祭壇で貴重品 **「狭間の魔石・赤（混沌・悪の証）」** を獲得。
*   **不合格時（閉鎖）**:
    - メッセージ: **「暗黒の霧が立ち込めている。混沌の闇を受け入れ、悪に身を染めた者でなければ、この邪悪な封印を解くことは叶わない……」** と表示し、進行を遮断する。

### 2.3 最深部「魔導士ヴァルグナ」撃破後の真の報酬分岐
*   アライメントの仕様上、1回の探索ランで「青の魔石」と「赤の魔石」を両方獲得することはできない。
*   プレイヤーは繰り返し受注可能（リピート）な「深層」および「最深部」を別アライメントビルドで複数回周回し、インベントリに「青の魔石」と「赤の魔石」の両方を集めて所持しておく必要がある。
*   両方の魔石を所持した状態で10Fボス「魔導士ヴァルグナ」を倒すと、消滅時に `check_delivery` ノードが起動する。魔石を両方納品することで、真のクリア報酬であるユニーク両手剣 **「デーモンバスター」** が解放される。

## 3. 未鑑定アイテムと鑑定システム (既存アセット混在・低確率レア調整)

「上層」「深層」「最深部」の宝箱やボス撃破報酬から、「未鑑定アイテム」がドロップする。これらはクリア報告用の専用クエストで鑑定し、装備品やスキルに置き換える。

### 3.1 未鑑定アセット（キーアイテム定義）
以下の4つの未鑑定キーアイテムを定義する。
*   `item_unidentified_weapon` (未鑑定の武器)
*   `item_unidentified_armor` (未鑑定の防具)
*   `item_unidentified_accessory` (未鑑定のアクセサリー)
*   `item_unidentified_magic` (未鑑定の魔導書)

### 3.2 鑑定報告クエスト (7064: 狭間の迷宮・探索報告) のフロー
*   **出現条件**: 上記の未鑑定アイテムを1つ以上インベントリに所持していること。
*   **ゲーム進行**:
    1.  受注すると、ギルドの「偏屈な鑑定士」との短い会話シナリオ（`text`）が開始。
    2.  `check_delivery` によって、所持している未鑑定アイテムが消費（没収）される。
    3.  即座にクエストクリア（`end_success`）となり、宿屋に帰還する。
    4.  クリア完了API（`/api/quest/complete`）が実行されると、消費されたアイテムに対応する以下のテーブルから、**新規の強力な武具と既存のアイテム（はずれ・コモン・消耗品含む）がランダムで1つ抽選され、インベントリに直接付与される**。

### 3.3 鑑定報酬プール（抽選テーブル）
周回ハクスラのゲームバランスを維持するため、新規実装される装備品の合算排出率を約10%に抑え、既存の武具や消耗品（はずれ・コモン枠）を90%の構成とします。
