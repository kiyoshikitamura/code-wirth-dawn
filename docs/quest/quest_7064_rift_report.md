## 1. クエスト概要

### 短文説明
```
[報告] 迷宮下層で獲得した「大魔道士の思念」をギルドへ報告せよ。
```

### 長文説明
```
「狭間の迷宮」下層の調査を終え、魔導士ヴァルグナの一時討伐に成功した。
しかし、バルガスによると、迷宮の奥底にはさらに深い「深淵」が存在し、真の脅威はまだ去っていないという。
深淵に入るための結界を破るには、下層の正義ルートで獲得した「大魔道士の思念」の魔力が必要となる。
思念の結晶を持たぬ者は深淵へ進む資格を失い、調査は打ち切られる。
また、もし闇の力を宿す「闇の宝珠」を併せて所持している場合、ギルドとの会話に不穏な影が落ちることになる。
```

## 3. シナリオノードフロー

```text
start_prep → start → text_vargas_01 → check_item_spirit
  ├── 所持していない場合 ──→ spirit_none (クエスト失敗：FAILURE)
  └── 所持している場合 ─────→ spirit_have → check_item_dark
                               ├── 「闇の宝珠」も所持している ──→ dark_have (闇分岐シナリオ) ─→ report_success
                               └── 「闇の宝珠」は所持していない ─→ dark_none (通常シナリオ) ─→ report_success
report_success → end_success
```

## 4. ノード詳細（抜粋・主要ギミック）

#### `check_item_spirit`（event / conditional branch）
**演出:** bg: bg_office
**処理:**
*   インベントリ内に **「330: 大魔道士の思念」** が存在するか判定。
*   所持している場合、`spirit_have` へ遷移。
*   所持していない場合、`spirit_none` へ遷移。

#### `spirit_none`（text）
**演出:** bg: bg_office, speaker: ギルド長バルガス
```text
「何だと？ ヴァルグナは退けたが、深淵の結界を解くための『大魔道士の思念』を手に入れられなかっただと？」
「くっ……それなしでは、これ以上調査を続けることは危険極まりない。残念だが、お前たちの調査任務はここまでだ。一旦頭を冷やし、下層をくまなく再捜索してこい！」
（※このノードの後、即座にクエスト失敗 [end_failure] となる）
```

#### `spirit_have`（text）
**演出:** bg: bg_office
```text
私たちはギルド長の机の上に、大魔道士の思念が込められた半透明の青白い水晶を置いた。結晶は美しい光を放ち、周囲の空気を浄化していく。
バルガスは息を呑み、それを手に取った。
「素晴らしい！ これこそが大魔道士の魂の結晶だ。これがあれば、深淵を阻む次元の結界を中和できる！」
```

#### `check_item_dark`（event / conditional branch）
**演出:** bg: bg_office
**処理:**
*   インベントリ内に **「329: 闇の宝珠」** が存在するか判定。
*   所持している場合、`dark_have` へ遷移。
*   所持していない場合、`dark_none` へ遷移。

#### `dark_have`（text）
**演出:** bg: bg_office, speaker: ギルド長バルガス
```text
「……待て。お前たちの荷物から、何か恐ろしい悪魔の気配を感じるぞ。その禍々しい黒紫色の球体は……まさか『闇の宝珠』か？」
バルガスは眉をひそめ、私たちを鋭く睨みつけた。
「それを何に使うつもりだ？ まさか悪魔の力を己の力にしようなどと考えているわけではあるまいな……？ 警告しておく。闇に魅せられた者は、最後には闇に喰われるぞ」
```

#### `dark_none`（text）
**演出:** bg: bg_office, speaker: ギルド長バルガス
```text
「よし、お前たちの正義の意思は本物のようだな。世界の命運は、この最後の『深淵』の調査にかかっている。大魔道士の思念を使い、迷宮最深部への門を開くぞ」
```

#### `report_success`（text）
**演出:** bg: bg_office
```text
バルガスから深淵調査の特別許可証を受け取る。世界の破滅を阻止するため、そして全ての決着をつけるため、私たちは深淵へと赴く決意を新たにした。
```
