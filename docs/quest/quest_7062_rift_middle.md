## 1. クエスト概要

### 短文説明
```
[探索] 迷宮の中層（B5F〜B7F）の罠を突破し、「青の宝珠」を守る魔皇を倒せ。
```

### 長文説明
```
「狭間の迷宮」の中層階、地下5階から地下7階の調査指示が下った。
中層は狂気と魅了の瘴気が濃くなっており、サキュバスやデーモンメイジといった魔法を駆使する悪魔が跋頭している。
また、仕掛けられたトラップ（HPダメージ）や虚無の落とし穴が多く、探索には細心の注意が必要となる。
地下7階の玉座で待ち受けるのは、迷宮中層を支配する冷酷な魔皇「深淵の狂皇 アスタロト」。
魔皇の牙城を崩し、第二の鍵である「青の宝珠」を奪取せよ。
```

## 3. シナリオノードフロー

```text
start_prep → start → maze_b5_start 
  → [B5F探索] B5F_battle_01 → B5F_treasure_01 → B5F_trap_01 → B5F_battle_02 → B5F_treasure_02
  → camp_01 (地下5階キャンプ)
  → [B6F探索] B6F_battle_01 → B6F_treasure_01 → B6F_trap_02 → B6F_battle_02 → B6F_treasure_02
  → camp_02 (地下6階キャンプ)
  → [B7F最深部] boss_gate → boss_battle
    ├─ win → boss_treasure (青の宝珠 確定獲得) → exit_gate → end_success
    └─ lose → end_failure
```
※各バトルノードの勝利先は、必ず直後の宝箱ノード（`type: 'treasure'`）に遷移する。
※道中には、プレイヤーの最大HPの10%ダメージを与えるトラップノード（`type: 'event'` / HP減少処理）が配置されている。

## 4. ノード詳細（抜粋・主要ギミック）

#### `B5F_trap_01`（event）
**演出:** bg: bg_rift_maze
```text
足元の石畳が不自然に沈み込んだ。直後、壁の隙間から次元の歪みが生み出した不可視の魔力刃が勢いよく飛び出してきた！
```
**処理:** パーティ全員のHPが最大HPの10%減少。

#### `boss_battle` (battle)
**演出:** bg: bg_rift_maze
**敵編成:** **【新規】深淵の狂皇 アスタロト (Lv.10)** ＆ **【新規】インプ (Lv.7) × 2**
*   **win:** `boss_treasure` へ遷移。
*   **lose:** `end_failure` へ遷移。

#### `boss_treasure` (treasure)
**演出:** bg: bg_rift_maze, layer: layer_rift_chest
*   **処理:** 特殊アクセサリー **「327: 青の宝珠」** を確定でインベントリに獲得。
*   **next:** `exit_gate` へ遷移。
