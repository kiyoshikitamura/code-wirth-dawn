## 3. シナリオノードフロー

```
start → set_bait → wait_tremor → sandworm_emerges → battle_sandworm
  ├─ 勝利 → worm_dead → end_success
  └─ 敗北 → end_failure
```

#### `start`（text）
```
交易商会から重大な依頼。砂漠の怪物「大砂虫」の討伐だ。
囮になる荷物を借り、出没地点へ向かう。
```

#### `set_bait`（text）
```
商隊が消えたというオアシス近くで、重い荷台を砂の上に置いた。
振動が地中に伝わるよう、太鼓のように荷台を打ち鳴らす。
```

#### `wait_tremor`（text）
```
……ゆっくりと、砂が波打ち始めた。
地面が揺れている——来る。
```

#### `sandworm_emerges`（text）
```
砂が爆発的に噴き上がった。
白い牙を剥いた巨大な顎が、空中に向かって開いた。
「デカい……！」
```
- 選択肢: 「戦う」→ `battle_sandworm`

#### `battle_sandworm`（battle）
**演出パラメータ:**
- **BGM**: `[要定義: 例 bgm_battle_normal]`

| 設定 | 値 |
|-----|-----|
| 敵グループ | `enemy_sandworm` |
| 敵名 | 大砂虫 |
| 備考 | HP非常に高い・DEF高め・特殊技あり（砂埃で手札制限など）のボス型 |

```
params: type:battle, enemy_group_id:[要定義: enemy_sandworm が含まれるグループ], next:worm_dead, fail:end_failure
```

#### `worm_dead`（text）
```
大砂虫が砂の上に倒れた。
その体は巨大な船ほどの大きさだった。
オアシスへの道が、再び開かれた。
```

#### `end_success`（end, result: success）
```
大砂虫討伐完了。交易商会から高額の報酬を受け取った。
```

#### `end_failure`（end, result: failure）
```
大砂虫の圧倒的な力の前に退かざるを得なかった。
```
