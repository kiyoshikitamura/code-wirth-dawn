## 1. クエスト概要

### 短文説明
```
国境警備。正体不明の武装集団を迎撃し、戦争の予兆を嗅ぎ取れ。
```

### 長文説明
```
国境警備任務に就いた君を待ち受けるのは、ただの野盗ではなかった。
統率の取れた武装集団の背後に蠢く大国の影。
ローランドとマルカンドの国境地帯で、戦争の気配が静かに忍び寄る。
```
