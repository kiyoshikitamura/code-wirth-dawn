## 1. クエスト概要

### 短文説明
```
[討伐] 迷宮最深部「深淵」の悪魔王を滅ぼし、伝説の聖剣を回収せよ。
```

### 長文説明
```
「狭間の迷宮」の最深部、理が崩壊した「次元の深淵」への突入を果たす。
このエリアには安全なキャンプなど一切存在せず、一瞬の油断が死を招く過酷な極限空間である。
徘徊するエネミーは、アークデーモンやグレーターデーモン、狂狂化したキメラなどの最上位悪魔・合成獣のみ。
道中の中ボスとして、次元の魔力と融合し真の魔人となった「魔導士ヴァルグナ（真）」が立ちはだかる。
そして最深部には、魔界の王にして絶対的な破壊の化身「破壊神 サタン」が降臨している。
人類の存亡を賭けた死闘を制し、魔王を討滅して、伝説の聖剣「デーモンスレイヤー」を勝ち取れ。
```
