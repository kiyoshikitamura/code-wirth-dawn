## 1. クエスト概要

### 短文説明
```
天使の軍勢が降臨。圧倒的な力の差に抗い、不死の傭兵王と出会う。
```

### 長文説明
```
夜刀の古代神殿で世界の真実——天上の神々による「大粛清」を知り、
東方の超大国「華龍神朝」へ足を踏み入れた。
だが目の前で、天使の軍勢が降り注ぎ、村が炎に呑まれていく。
絶望的な戦力差の中で刃を向けた先に待っていたのは、
生きる伝説——「不死の傭兵王」ヴォルグとの邂逅だった。
```
