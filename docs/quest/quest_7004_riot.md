### ノード詳細

#### `start`（type: text）
**BGM**: `bgm_quest_calm`
**背景画像**: `bg_guild`
**SE**: `se_quest_accept`

**テキスト:**
```
自治会の使いが酒場に駆け込んできた。息を切らしている。

「頼む、すぐに来てくれ！ 広場に市民が集まって、倉庫を襲うって騒いでるんだ！
　自警団じゃ手が足りない。傭兵に頼むしかないんだよ……！」

男の顔は蒼白だった。
自治会の仕事か。報酬は出る——が、後味の悪い類の仕事だ。
```

---

#### `briefing`（type: text）
**背景画像**: `bg_guild`

**テキスト:**
```
自治会の長老が、しわがれた声で状況を説明した。

「食料の配給が三日前から止まっている。原因は流通の乱れだが、
　民衆はそうは思っておらん。我々が溜め込んでいると疑っておるのだ」

老人は目を伏せた。
「演説をしている男がいる。あいつが煽動者だ。
　あいつを黙らせれば、残りは散る……そのはずだ」

散るかどうかは、こちらの暴力の程度によるだろう。
```

---

#### `approach_square`（type: text）
**BGM**: `bgm_quest_tense`
**背景画像**: `bg_road_day`

**テキスト:**
```
広場に近づくと、怒号が聞こえてきた。
百人近い市民が集まっている。女性や子供の姿も見える。

木箱の上に立った痩せぎすの男が、拳を振り上げて叫んでいる。
「俺たちの食い物はどこへ消えた！？ 奴らの倉庫にあるんだ！
　黙って飢え死にしろってのか！？」

群衆が呼応する。空気が揺れている。
```

---

**背景画像**: `bg_road_day`

**テキスト:**
```
指導者の男がこちらに気づいた。
目に怒りと、ほんの僅かな恐怖が混じっている。

「傭兵か……！ 自治会の犬め！
　俺たちは腹が減ってるだけだ！ それが罪か！？」

男は鎌を握りしめている。後ろの群衆も、手に手に棒切れや農具を持っている。
話し合いで済む段階は、とうに過ぎているらしい。
```

「……配給は再開するのか？」
