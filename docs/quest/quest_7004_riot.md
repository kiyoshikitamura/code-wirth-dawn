## 3. シナリオノードフロー

```
start → locate_mob → battle_mob
  ├─ 勝利 → disperse → end_success
  └─ 敗北 → end_failure
```

#### `start`（text）
```
自治会の依頼。食料を求めて街頭に集まろうとしている市民のデモを未然に鎮圧する。
報酬は高い。後味は……考えないことにした。
```

#### `locate_mob`（text）
```
情報通りの場所に人だかりができ始めていた。
木箱の上に立ち演説しようとしている男が指導者だろう。
```

#### `battle_mob`（battle）
**演出パラメータ:**
- **BGM**: `[要定義: 例 bgm_battle_normal]`

| 設定 | 値 |
|-----|-----|
| 敵グループ | `enemy_mob` |
| 敵名 | 暴徒（飢えた市民） |
| 備考 | ATK低め・DEF低め。数で圧してくる構成を想定 |

```
params: type:battle, enemy_group_id:[要定義: enemy_mob が含まれるグループ], next:disperse, fail:end_failure
```

#### `disperse`（text）
```
指導者が倒れると、群衆はたちまち蜘蛛の子を散らすように逃げていった。
自治会の男が後ろから「ご苦労」と言った。その顔を見たくなかった。
```

#### `end_success`（end, result: success）
```
鎮圧完了。報酬を受け取った。胸に残るものは……捨てておく。
```

#### `end_failure`（end, result: failure）
```
暴徒の勢いに押し切られた。デモは強行された。自治会からの報酬はない。
```
