## 1. クエスト概要

### 短文説明
```
都を蝕む砂の病。不可視の呪いに守られた「禁忌の王墓」で、死の知略に挑め。
```

### 長文説明
```
マルカンドの地下深く、数千年間呪いに守られてきた「無名王の王墓」。
一歩進むたびに命を削る「呪い」と「言葉の罠」が待ち受ける。
謎を解き明かし、最深部に眠る王の呪いを止めろ。
```

**テキスト:**
```

砕けた瞬間、広間全体が震えた。

三千年の魔力が、一瞬で消え去った。

刀身に砂の模様が走っている。重い刃だ。
```
