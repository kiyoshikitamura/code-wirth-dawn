## 1. クエスト概要

### 短文説明
```
[探索] 迷宮の上層（B1F〜B4F）を攻略し、守護者が抱える「赤の宝珠」を回収せよ。
```

### 長文説明
```
「狭間の迷宮」の上層階、地下1階から地下4階までの先遣調査が開始された。
迷宮の内部は次元が歪んだ迷路のようになっており、下級の悪魔やゾンビ、スケルトンが徘徊している。
道中には疲弊した冒険者のためのキャンプが3箇所設置されている。
地下4階の最深部に君臨する、悪魔の力で狂暴化した「スケルトンキング」を討伐し、第一の鍵となる「赤の宝珠」を回収せよ。
```

## 3. シナリオノードフロー

```text
start_prep → start → maze_b1_start 
  → [B1F探索] B1F_battle_01 → B1F_treasure_01 → B1F_battle_02 → B1F_treasure_02
  → camp_01 (地下1階キャンプ)
  → [B2F探索] B2F_battle_01 → B2F_treasure_01 → B2F_battle_02 → B2F_treasure_02
  → camp_02 (地下2階キャンプ)
  → [B3F探索] B3F_battle_01 → B3F_treasure_01 → B3F_battle_02 → B3F_treasure_02
  → camp_03 (地下3階キャンプ)
  → [B4F最深部] boss_gate → boss_battle
    ├─ win → boss_treasure (赤の宝珠 確定獲得) → exit_gate → end_success
    └─ lose → end_failure
```
※各バトルノードの勝利先は、必ず直後の宝箱ノード（`type: 'treasure'`）に遷移する。
※キャンプノードでは、「帰還する（クエスト達成①）」を選ぶと、その時点でクリア（達成①）扱いとなり、それまでの経験値やアイテムを持ち帰って終了する（ただし次のクエストは解放されない）。「探索を続ける」を選ぶと、次の階層へ進む。

## 4. ノード詳細（抜粋・主要ギミック）

#### `camp_01` `camp_02` `camp_03`（choice / camp）
**演出:** bg: bg_rift_camp
*   **選択肢1:** 「装備を変更する / アイテムを使用する」（装備変更・消耗品使用メニューを表示）
*   **選択肢2:** 「街へ一時帰還する」（その場で安全に街に帰還し、クエスト達成①として終了）
*   **選択肢3:** 「さらに深部へ探索を進める」（次の階層の開始ノードへ遷移）

#### `B1F_battle_01` (battle)
**演出:** bg: bg_rift_maze
**敵編成:** `pool_rift_upper` からランダムに選出。
*   **win:** `B1F_treasure_01` へ遷移。
*   **lose:** `end_failure` へ遷移。

#### `B1F_treasure_01` (treasure)
**演出:** bg: bg_rift_maze, layer: layer_rift_chest
*   **処理:** 上層ドロップテーブルからアイテム獲得判定。
*   **next:** `B1F_battle_02` へ遷移。

#### `boss_battle` (battle)
**演出:** bg: bg_rift_maze
**敵編成:** **【既存】スケルトンキング (Lv.8)** ＆ **【新規】インプ (Lv.7) × 2**
*   **win:** `boss_treasure` へ遷移。
*   **lose:** `end_failure` へ遷移。

#### `boss_treasure` (treasure)
**演出:** bg: bg_rift_maze, layer: layer_rift_chest
*   **処理:** 特殊アクセサリー **「326: 赤の宝珠」** を確定でインベントリに獲得。
*   **next:** `exit_gate` へ遷移。
