## 1. クエスト概要

### 短文説明
```
龍京を蝕む大天使ラファエル。癒しの名の下に、都は静かに朽ちていく。
```

### 長文説明
```
イスハークでウリエルを退けた後、華龍の首都・龍京へ急行した。
だが龍京を襲っていたのは炎ではなく——静寂だった。
大天使ラファエルの「癒し」の力が、生きとし生けるものの力を奪い、
都市全体を深い眠りに沈めようとしている。
```
