## 1. クエスト概要

### 短文説明
```
世界の中心「虚無の塔」。華龍の四方に座す四神を討ち、神の残酷な遊戯を終わらせよ。
```

### 長文説明
```
塔の頂上には願いを叶える楽園があるとされる。
しかし、神の結界により近づく事はできない。解くためには四神討伐が必要だ。
すべては、高みで退屈を貪る「神」が仕掛けた遊戯であった。
```
