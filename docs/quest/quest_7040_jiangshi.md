## 1. クエスト説明

### 短文（ボード表示・40字以内）
```
霊山を徘徊するキョンシーを、護符の力で鎮めよ。
```

### 長文（詳細モーダル）
```
華龍神朝の道士から依頼が届いた。
近年、霊山に葬られた者たちが「跳屍」——いわゆるキョンシーとなって歩き回り、
参拝者を脅かしているという。
道士は「これは自然の乱れが原因で、普通の武器では仕留められない」と言うが、
特製の護符を複数枚準備してくれるので、それを使えば制圧できるはずだとのこと。
護符の使い方は——「額に貼れば止まる」。
その後は道士が引き取って埋葬し直す。
奇妙な依頼だが、報酬は誠実だ。
```
