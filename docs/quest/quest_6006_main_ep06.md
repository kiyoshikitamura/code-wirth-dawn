## 1. クエスト概要

### 短文説明
```
敗残兵として密林を逃走。追手を振り切り、夜刀神国の関所を目指せ。
```

### 長文説明
```
ガウェインを失い、敗残兵の群れに紛れて前線を離脱した。
王国の残党狩りが迫る密林地帯を、傷だらけの体で駆け抜ける。
生き延びることだけが、今の君に残された唯一の戦い方だ。
```
