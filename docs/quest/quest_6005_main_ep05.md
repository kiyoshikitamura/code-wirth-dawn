## 1. クエスト概要

### 短文説明
```
全面戦争。津波のような王国軍を前に、老騎士が最後の壁となる。
```

### 長文説明
```
開戦の口火が切られた。
ローランド王国軍の本隊がマルカンドの前線拠点に全面侵攻を開始する。
津波のように押し寄せる鉄の波の中で、
老騎士ガウェインは君を守るために最後の盾となる。
大義という名の下に散る命の重さを背負い、孤独な旅が始まる。
```
