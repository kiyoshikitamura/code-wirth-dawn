## 1. クエスト概要

### 短文説明（クエストボード表示・40文字以内）
```
廃墟に巣食うゴブリンを退けて、金庫から物資を回収せよ。
```

### 長文説明（詳細モーダル・フレーバーテキスト）
```
かつて栄えた街区が崩落して久しい。
ギルドは廃墟の地下に封じられた金庫の存在を掴んでいる。
問題は、その廃墟に小鬼（ゴブリン）が巣食っていることだ。
奴らは知恵こそ乏しいが、数が多く縄張り意識が強い。
金庫を開けるには廃墟の中へ踏み込み、
ゴブリンの群れを排除しながら奥へ進む必要がある。
回収した物資は必ずギルドへ届けること。
そうすれば、追加報酬のアイテムが一緒に支払われる。
```
