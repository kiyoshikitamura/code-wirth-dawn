## 1. クエスト概要

### 短文説明
```
炎に包まれたイスハーク。大天使ウリエルの裁きに、ヴォルグと共に立ち向かえ。
```

### 長文説明
```
ヴォルグと共にマルカンドの首都イスハークへ急行した。
だが黄金の都はすでに炎に包まれていた。
大天使ウリエル——炎を司る審判者が、この都市を焼き尽くそうとしている。
逃げ惑う民を守るため、炎の中へ踏み込む。
```
