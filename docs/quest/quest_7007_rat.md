```
[駆除] 地下水路の巨大ネズミの群れを2連戦で殲滅せよ。
```

## 3. シナリオノード構成

### 全体フロー

```text
start
  └─[続ける]→ enter_sewers
       └─[続ける]→ first_encounter
            └─[戦う]→ battle_rats_01
                 ├─ [勝利] → deeper_sewers
                 │    └─[続ける]→ nest_approach
                 │         └─[戦う]→ battle_rats_02
                 │              ├─ [勝利] → destroy_nest → end_success
                 │              └─ [敗北] → end_failure
                 └─ [敗北] → end_failure
```

### ノード詳細

#### `start`（type: text）
**BGM**: `bgm_quest_calm`
**背景画像**: `bg_guild`
**SE**: `se_quest_accept`

**テキスト:**
```
自治会の衛生官が、汚れた手袋を外しながら説明した。

「地下水路のネズミが尋常じゃない大きさに育っている。
　犬ほどもある個体が確認された。噛まれた清掃員が三人、熱を出して寝込んでる」

衛生官は顔をしかめた。
「このまま放置すれば疫病だ。巣を見つけて焼き払ってくれ。
　松明と油は支給する。あとは——腕と度胸が必要だ」
```

---

#### `enter_sewers`（type: text）
**BGM**: `bgm_quest_tense`
**背景画像**: `bg_catacombs`

**テキスト:**
```
地下水路の入口。鉄格子を外すと、湿った空気と腐敗臭が吹き上げてきた。
松明の灯りが水面に揺れる。

壁一面に無数の爪痕が刻まれている。
水路の水は黒く濁り、何かが水面下を横切った。

「……相当な数だ」

足元の水が膝まで達する。進むたびに、黒い影がちらちらと動く。
```

---

#### `first_encounter`（type: text）
**背景画像**: `bg_catacombs`

**テキスト:**
```
角を曲がった瞬間、赤い目が一斉にこちらを向いた。
犬ほどもある巨大ネズミが、水路の上に群がっている。

黄ばんだ牙を剥き、甲高い鳴き声を上げた。
足元の水が、ネズミたちの動きで波立っている。
```
**選択肢:**
| ラベル | 次ノード |
|--------|---------|
| 戦う | `battle_rats_01` |

---

#### `battle_rats_01`（type: battle）
**BGM**: `bgm_battle_normal`

| 設定 | 値 |
|-----|-----|
| 敵グループ | `enemy_giant_rat` x3 |
| 敵名 | 巨大ネズミ（群） |
| 備考 | HP低め・数で圧す。毒針スキルあり |

**params:**
```
type:battle, enemy_group_id:grp_rat_swarm, next:deeper_sewers, fail:end_failure
```

---

#### `deeper_sewers`（type: text）
**背景画像**: `bg_catacombs`

**テキスト:**
```
最初の群れを蹴散らした。
水路の奥から、さらに甲高い鳴き声が反響してくる。

仕留めたネズミの体を見ると、腹が異様に膨れている。
何かを食べて急激に成長したらしい。
——汚水に混じる薬品か何かが原因だろうか。

巣がある場所はもっと深い。先に進む。
```

---

#### `nest_approach`（type: text）
**背景画像**: `bg_catacombs`

**テキスト:**
```
水路の最奥部。天井が低くなり、身を屈めなければならない。
松明の灯りに照らされたのは——異様な光景だった。

壁一面を覆う巨大な巣。布切れ、骨、金属片を編み込んだ塊。
その中央に、ひときわ大きな個体が鎮座している。
子ネズミたちが周囲でうごめいている。

親ネズミがこちらに気づいた。牙を剥き、尾で地面を叩く。
```
**選択肢:**
| ラベル | 次ノード |
|--------|---------|
| 巣ごと焼き払う | `battle_rats_02` |

---

#### `battle_rats_02`（type: battle）
**BGM**: `bgm_battle_normal`

| 設定 | 値 |
|-----|-----|
| 敵グループ | `enemy_giant_rat_alpha` x1 + `enemy_giant_rat` x2 |
| 敵名 | ネズミの女王 + 巨大ネズミ |
| 備考 | 第2戦強化版。女王はHP高め |

**params:**
```
type:battle, enemy_group_id:grp_rat_nest, next:destroy_nest, fail:end_failure
```

---

#### `destroy_nest`（type: text）
**BGM**: `bgm_quest_calm`
**背景画像**: `bg_catacombs`

**テキスト:**
```
巣の主を倒した。
残った子ネズミたちが蜘蛛の子を散らすように逃げていく。

支給された油を巣にかけ、松明で火をつけた。
湿った巣がじゅうじゅうと音を立てて燃えていく。

黒い煙が水路を満たした。目が痛む。
だが、これで終わりだ。水路がゆっくりと流れを取り戻す。
```

---

#### `end_success`（type: end, result: success）
**SE**: `se_quest_success`

**テキスト:**
```
害獣駆除完了。地上に這い出ると、外の空気が初めて美味く感じた。
自治会に報告すると、衛生官が深々と頭を下げた。

「助かった。これで疫病は防げる。
　……水路の中で何を見たかは、あまり触れ回らないでくれ。
　市民が怖がるからな」

報酬を受け取った。靴は捨てることにした。
```

---

#### `end_failure`（type: end, result: failure）

**テキスト:**
```
ネズミの群れに圧倒され、水路から這い出した。
全身に噛み傷。服は汚水まみれ。
自治会の衛生官は、別の傭兵を探すと言った。
```
