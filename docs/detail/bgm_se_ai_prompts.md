# Wirth Dawn 音源生成プロンプト集（AI用）

ブラウザ版Gemini等の音楽・効果音生成AIにそのままコピー＆ペーストして使えるプロンプト案です。
Wirth Dawnの**「JRPG」「退廃的」「TRPGライク」**という世界観のベース指示を最初に含めることで、統一感のある音源が生成されやすくなります。

## 🎵 BGM（全11曲）

BGMは長めの尺（1〜2分以上）で、ループ（繰り返し）再生しても違和感がないように生成するのがコツです。

### 0. ベースプロンプト（全体の世界観指定）
AIに曲のイメージを掴ませるため、以下のテキストをBGMプロンプトの先頭に追加すると効果的です。
> `物語主導の王道JRPG（和製ロールプレイングゲーム）のBGMを作成してください。ボーカル（歌声）は一切含めず、完全なインストゥルメンタル楽曲にしてください。過酷な世界観（TRPGライク・少し退廃的）でありながらも、これから未知なる冒険へと旅立つワクワク感や、プレイヤーのテンションを押し上げるような明るさを持ったキャッチーなメロディの楽曲を希望します。ループ再生に適した構成にしてください。`

## 🔊 SE（効果音）

SEは、1秒〜3秒程度の「短い音」として出力させる必要があります。Gemini等に**「数秒だけの単発の効果音を作成して」**と強く念押ししてください。
※ 音楽生成AIの仕様上、どうしても最低30秒で出力されてしまうことがあります。その場合はプロンプトの末尾に `[End]` や `[Stop]` などの終了タグを入れるか、**「音が鳴った直後に完全な無音（Silence）にして終了させてください」**と指定してください。

### 0. SE用ベースプロンプト（全体共通指定）
AIが「歌」として誤認識してボーカル付きの楽曲を出力してしまうのを防ぐため、以下のテキストを**各SEプロンプトの先頭に必ず追加**して生成を行ってください。
> `【ボーカル禁止・歌声不可】人間の声、ボーカル、コーラス、ラップなどの音声は絶対に含めないでください。完全なインストゥルメンタル（非ボーカル）であり、かつ純粋な環境音や効果音（Foley Sound Effect）として出力してください。`

### 生成したファイルの扱い方
出力されたファイルはすべてダウンロードし、**ファイルの拡張子より前の部分を、上記の表の左側にある名前にリネーム**してください。（例：ダウンロードした音楽を `bgm_battle.mp3` に名前変更）。
その後、先ほど作成したフォルダ `_raw_audio\bgm` や `se` に配置していってください。
