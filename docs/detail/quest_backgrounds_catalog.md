# クエスト用背景画像 カタログ（Quest Backgrounds Catalog）

クエストデータの作成時に、`script_data` 内の `bg_image` 等へ指定するための背景画像の一覧です。
すべての画像は `public/images/quests/` ディレクトリ配下に `.png` 形式で保存されています。（※実際のクエストデータ定義時には、拡張子に応じたパス `/img/quests/[slug].png` またはウェブ仕様に合わせて調整して使用してください）

## 1. 拠点・室内系 (Towns & Interiors)
クエストの受注場所やクライアントとの密談、室内戦闘などに使用します。

| ファイル名 | カテゴリ | 概要 |
| --- | --- | --- |
| `bg_guild.png` | 室内 | 冒険者ギルド・配達組合などの活気ある木造の空間 |
| `bg_tavern_day.png` | 室内 | 昼間の明るい酒場・食事処 |
| `bg_tavern_night.png` | 室内 | 夜の薄暗い酒場・黄昏亭・密談向けの空間 |
| `bg_office.png` | 室内 | 領主の館、将軍の執務室などの厳格で少し豪華な部屋 |
| `bg_shop.png` | 室内 | 武器屋、よろず屋。アイテムが並ぶ雑然とした室内 |
| `bg_church.png` | 室内 | 教会の大聖堂。ステンドグラスから差し込む神聖な光 |
| `bg_slum.png` | 室内外 | スラム街、裏路地。治安の悪そうな暗い路地裏 |

| `bg_forest_day.png` | 屋外 | 木漏れ日の差し込む美しい昼の森 |

| ファイル名 | カテゴリ | 概要 |
| --- | --- | --- |
| `bg_catacombs.png` | 迷宮 | 地下墓所、安置所。ロウソクと石壁に囲まれた不気味な空間 |
