### 画像生成時のガイドライン (JRPGアニメ調仕様)
旧仕様における「ダークファンタジー」「油彩風の重厚な塗り」「グランジ・泥臭さ」といった指示は**完全に廃止**され、現在は**JRPGアニメ調（セルルック）**を基本デザインとしています。
画像生成エージェントは、以下の要素をプロンプトに組み込んで画像を生成してください：

- `JRPG anime art style item icon, highly detailed, beautiful cell shading, clear and vibrant colors` (JRPGアニメ風のアイテムアイコン、美しいセルルック、鮮明でクリアな色彩)
- `single isolated object, centered composition, simple dark or solid background` (独立した単一のオブジェクト、中央配置、アイテムが目立つシンプルな背景)
- `(no text, no letters, no UI elements, no characters)` (文字やUI、キャラクターの混入を厳格に排除)
- グロテスクな表現（人皮・血・汚すぎる泥など）は避け、JRPGらしい「魔法のオーラ」「光のパーティクル」を用いたファンタジー表現に置き換えること（AIの安全フィルター制限を回避する目的も含みます）。
