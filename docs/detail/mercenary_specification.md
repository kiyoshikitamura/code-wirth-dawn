## 1. 50体追加計画マトリクス

**国家・所属別（各10体）**:
1. ローラン聖帝国（Roland）
2. 砂塵の王国マルカンド（Markand）
3. 夜刀神国（Yato）
4. 華龍神朝（Karyu）
5. フリーランス/英雄（Freelance/Neutral）

**レベル帯・価格設定**:
- **序盤用（Lv1〜5）**: 雇用金 6,000G〜10,000G
- **中盤用（Lv10前後）**: 雇用金 15,000G前後
- **英雄クラス（Lv20〜30）**: 雇用金 25,000G〜35,000G

**AIロールの分散**:
- `striker`（攻撃特化）: 20体
- `guardian`（防御系）: 15体
- `medic`（回復特化）: 15体

## 2. 投入用CSVのフォーマット定義

| カラム名 | 型 | 説明 |
|---|---|---|
| `slug` | string | 一意の識別子（例: `sys_merc_roland_01`） |
| `name` | string | 傭兵の名前（UI破綻防止のため12文字以内推奨） |
| `gender` | string | `Male` / `Female` / `Unknown` |
| `job_class` | string | 職業または二つ名 |
| `faction` | string | 所属国家（Roland, Markand, Yato, Karyu, Neutral） |
| `level` | number | 基本レベル（1〜30） |
| `durability` | number | 寿命/耐久度（基本は30〜50） |
| `base_cost` | number | 雇用金額。基本計算式: `5000 + (Lv * 1000)` |
| `ai_role` | string | AIの行動方針（`striker`, `guardian`, `medic`） |
| `inject_cards` | JSON | デッキに追加されるカードIDの配列文字列 `[1, 5, 12]` |
| `flavor_text` | string | 酒場で表示される背景設定・口上（60文字以内） |
| `visual_prompt` | string | アイコン画像生成AI向けのプロンプト指示 |
