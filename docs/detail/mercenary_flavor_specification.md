# 傭兵NPC フレーバー＆ビジュアル仕様書（npcs.csv準拠・正式版）

本仕様書は、`src/data/csv/npcs.csv` の53体（一般50体＋ゲスト3体）を正として、
通り名（epithet）と名前（name）を分離した上で、各NPCのフレーバーテキスト・ビジュアルプロンプト・スキル構成を定義したものです。

---

## スキル正常化について

既存CSVでは、多くの回復役のカードに `2001`（強撃NPC）が設定されており、実際には物理攻撃を行っていました。
本仕様書および更新後のCSVでは、以下のように訂正済みです。

| 修正対象 | 修正前 | 修正後 | 理由 |
|---|---|---|---|
| エレナ, アンナ, サクラ, 神主, メイド, 吟遊詩人 | `2001` (強撃NPC) | `1003` (ヒール) | 回復役のはずが攻撃していた |
| 猫 | `2001` | `2000` (応急手当) | 癒やし要員だが攻撃していた |
| レオ, 戦闘料理人ワン | `2001` | `1003` | 回復スキル枠のはずが攻撃だった |
| 曲刀使い | `1001`x2 (斬撃) | `1012`x2 (曲刀術) | フレーバーと不一致 |
| 槍術師範代 | `1001`x2 | `1015`x2 (白銀の槍) | テーマ武器不一致 |
| ランプ使いの老人 | `2010`x2 | `2010`+`2080` (魔人召喚) | 召喚テーマ不一致 |
| イカサマ師 | `2099` (逃走) | `2089` (黄金のサイコロ) | 賭博テーマ不一致 |
| 陰陽師アベ | `2010` (錬金火球) | `2040` (式神使役) | 陰陽術テーマ不一致 |

## 表示ルール

- **バトル画面・ステータスバー**: `name`（例：エレナ）のみ表示
- **酒場の詳細パネル**: `${epithet} ${name}`（例：見習い僧侶 エレナ）と連結
- **リスト表示のサブテキスト**: `epithet` が設定されていれば通り名を表示、なければ `job_class` の日本語訳を表示
