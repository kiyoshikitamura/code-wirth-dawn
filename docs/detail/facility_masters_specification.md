# 施設マスター 名称・配置・セリフ仕様書

『Code: Wirth-Dawn』における、各拠点施設の主人（NPC）の名称、デザインテイスト、配置拠点、およびプレイヤーの「名声（Fame）別セリフ」の統合仕様書です。
UI（StatusModalやScenarioEngine等）での改行を防ぐため、キャラ名称は全角3〜5文字程度に短縮・最適化されています。

## 🎨 画像生成における共通ルール (Visual Pipeline Rules)
1. **テイストと塗り方**: 厚塗りは避け、日本人に馴染みのあるクリアなJRPG風アニメ調セルルック。
2. **構図とキャラサイズ**: モバイルUI向け「バストアップ（Medium close-up）」を基本とし、表情を明確に描画。背景にファンタジーの旅情を取り入れる。
3. **テキスト生成の禁止**: `(no text, no letters)` 等を含め、文字混入を固く禁じる。
4. **没画像の削除**: ボツ生成画像はディレクトリ圧迫を防ぐため削除徹底。

## 🗺️ 拠点配置の基本ルール

初期拠点の「名もなき旅人の拠所」には固有マスターである「ダン」が配置され、以降の20拠点は各地方内のマスター1〜5がエリア順に配置されます。

## 🗣️ 名声度（Fame）別セリフの4段階定義
名声度に応じて以下の4段階でセリフが変化します。宿屋と酒場は同一キャラクターが担当しますが、施設ごとの独立した台詞を定義します。

1. **名声高 (High)**: 【尊敬と称賛がある台詞】英雄としての対応。
2. **通常 (Normal)**: 【通常の対応】標準的な客扱い。
3. **名声低 (Low)**: 【悪人に対する対応や蔑み】悪評を嫌う冷たい態度、警戒。
4. **出禁 (Lowest)**: 【出入り禁止の対応】本格的な怒りや追い出し（※施設利用不可）。
