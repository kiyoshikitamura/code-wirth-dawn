# Hub (Inn) Top Screen UI & NPC Dialogue System

## 1. 概要 (Overview)
本仕様書は、プレイヤーの拠点（宿屋/名も無き旅人の拠所を含む全20拠点）のトップ画面UI、および「名声」と「情勢（繁栄度）」によって変化する静的/動的なNPC対話システムについて定義する。

## 2. 画面構成 (Screen Layout)
画面は以下の4つの主要エリアで構成される（モバイルレイアウト 390x844 を基準とする）。

### 2.1 グローバル情報ヘッダー
- **プレイヤー情報**: アバター画像右下に `Lv.X` のバッジを付与。HP/APバー下部に `名声: X` を表示。
- **Vitality警告**: `Vitality <= 20` の場合、アイコンとテキストが「赤色で点滅（Pulse）」する視覚的警告を行う。
- **世界情勢**: 聖界暦（経過日数換算）、現在地名、統治国家、現在の情勢（繁栄度ベース）を表示。情勢に応じたフレーバーテキストをイタリック体で併記。
- マルチセグメントバー（または情勢エリア）の上部には明示的に「世界の覇権」というラベルテキストを追加する。

### 2.2 メインビジュアルエリア
- 背景画像を用いた没入感のあるエリア。
- 右下に以下の**アイコンのみの円形ボタン**をコンテキストメニューとして並べて配置する。
  - **歴史（BookOpen）**: メインストーリーや過去の履歴を参照。
  - **街の噂話（MessageSquare）**: タップで「噂話モーダル」を起動。滞在拠点の情勢に応じた複数のフレーバーテキスト・リストを表示。

### 2.3 施設ナビゲーショングリッド
- 2列×4行のグリッドで構成。
- `1. 宿屋`, `2. 道具屋`, `3. 酒場`, `4. 神殿`, `5. ギルド` はタップ時に「施設NPC対話モーダル」を開く。
- `6. マップ`, `7. ステータス`, `8. 設定` は直接該当画面やモーダルへ遷移する。

### 2.4 クリエイターズ工房バナー (Hub限定)
- プレイヤーの現在地が「名も無き旅人の拠所」である場合のみ、ナビゲーショングリッド下部にグラデーション背景の専用バナーを表示。
- バナーテキストには「クリエイターズ工房」のみを表示し、不要なアイコンやテキストを排除する。
- タップで上寄せ表示の「工房モーダル（アセット作成・作品管理など）」を起動。直接エディタへは遷移させない。

## 3. NPC対話システム (NPC Dialogue System)
各施設（宿屋〜ギルド）を利用する際、即座に画面遷移するのではなく、画面下部からスライドアップする「NPC対話モーダル」を挟み、ロールプレイの没入感を提供する。

### 3.1 モーダル構成と配置
- **配置**: すべてのモーダル（NpcDialogModal, RumorsModal, WorkshopModal, HistoryModal等）は、ヘッダーを隠さないように配慮しつつ**画面上端（Headerのすぐ下）**に配置する（items-start, pt-28などを利用）。
- **背景**: 半透明ダークパネル。進入アニメーションは下からのスライドインではなく、**上からのスライドイン（slide-in-from-top）**またはフェードインを採用。
- **ヘッダー**: NPCの役職（例: `[宿屋の主人]`）、NPC名、NPCアイコン画像。
- **ダイアログ**: NPCの台詞。
- **アクション**: 最下部に各施設に応じた個別のアクションラベルと挙動を持つメインボタンを配置する。
  - **宿屋**: 「休息する」 -> ゴールドを消費し、HP/APを全快させる。コストは情勢（Prosperity）に応じて変動する（繁栄(4以上): 100G, 通常(3): 200G, 衰退/崩壊(2以下): 300G）。所持金不足時はボタンを `disabled` とし「ゴールドが不足しています」と表示。
  - **道具屋**: 「品揃えを見る」 -> ショップUI（ShopModal）起動
  - **酒場**: 「冒険者を探す」 -> 酒場UI（TavernModal）起動
  - **神殿**: 「礼拝堂に行く」 -> 祈りUI（PrayerModal）起動
  - **ギルド**: 「依頼を見る」 -> ギルド画面（/guild）へ遷移

### 3.2 動的台詞分岐ロジック
台詞は固定ではなく、以下の優先順位でプレイヤーの「名声」と所在地の「情勢」によって分岐する。

1. **名声（Renown）による分岐**:
   - `Renown >= 500`: 英雄に対する歓迎、尊敬の声。 (例: 「おお、噂の英雄殿！当施設をぜひご利用ください！」)
   - `Renown < 0`: 悪名に対する警戒、または一部施設での裏取引の示唆。
2. **情勢（Prosperity）による分岐**:
   - `Prosperity <= 1` (崩壊/衰退): 物資不足、略奪への恐怖、悲観的な台詞。
   - `Prosperity >= 4` (繁栄): 景気の良さ、豊富な物資をアピールする台詞。
3. **デフォルト（通常）**: 上記条件に当てはまらない汎用的な台詞。

## 4. デザイン要件
- テーマカラー: `Slate 950 (#020617)` をベースとし、アクセントカラーに `Gold (#D4AF37)` および `Amber-500` を使用。
- モーダルやボタンのタップ時には `active:scale-95` のフィードバックを実装すること。
